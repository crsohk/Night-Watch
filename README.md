# Night Watch (v3.4.0)

밤당직(19:00 → 익일 07:00) 전용 오프라인 아이폰 웹앱. UI는 전면 영문(Lato),
베이지(사막의 낮) 테마, 스크롤 없는 한 화면, 1.5배 세로의 가로 배터리 게이지.

## 1. GitHub Pages 배포

1. github.com → New repository (예: `nightwatch`) → Create
2. Add file → Upload files → 이 폴더 안의 모든 파일·폴더 업로드
3. Settings → Pages → Branch `main` / `(root)` → Save
4. `https://<아이디>.github.io/nightwatch/` 확인

이미 배포했다면 덮어쓰기만 하면 됩니다. sw.js VERSION이 v3.4.0이라 자동 갱신됩니다.

> 참고: 개인 전화번호는 README에 표기하지 않습니다. ANE 번호는 앱 안
> EDIT에서 직접 입력하세요 (앱 데이터는 기기에만 저장됩니다).

## 2. 아이폰 설치

Safari 접속(첫 1회 온라인) → 공유(⬆︎) → **홈 화면에 추가**. 이후 오프라인 완전 동작,
날씨만 온라인 시 갱신.

## 3. 화면 (한 화면, 스크롤 없음)

- 가로 배터리: 왼쪽 끝 NOW(병원 현재 날씨) → 오른쪽 끝 END(종료 시각 날씨),
  중앙에 소화율 %. 아래 ELAPSED / REMAINING.
- START 직후 무환자나무 인트로(잎이 2장씩 열림, 약 5초)가 지나가고 당직 화면으로 전환.
- CALL +1: 잎 왼쪽 끝. 잎 오른쪽 끝의 UNDO 원형 버튼 = 실수 취소.
  콜 시각 로그·조기 종료는 하단 바의 ≡ 버튼(당직 중에만 표시).
- 문구 카드: 시간대별 응원 문구(듄 + 병원 유머, 영문).
- CALL LINE: ER · ANE · OR · EICU · GW1 · GW2 · ICU · PA · CR · UGI · HBP · VAS. 탭 = 즉시 발신.
  **전화번호는 화면에 표시되지 않습니다** (이름/라벨만).
- 작은 기종은 자동 압축(문구는 최후 단계까지 유지)으로 항상 한 화면.

## 4. CALL LINE 설정 (EDIT)

고정 라인(기본 등록): ER 031-787-3001 · OR 031-787-3355 · EICU 031-787-3700,
그리고 **GW1 · GW2 병동 당직폰(v3.4.0부터 고정, 번호는 README에 표기하지 않음)**.
ANE는 기본값 없음 - EDIT에서 직접 입력. 고정 라인도 EDIT에서 언제든 수정 가능.

ICU · PA · CR · UGI · HBP · VAS (당직마다 교체):
- **자동 초기화 (v3.3.4)**: 당직 종료 후(저장 다음의 첫 07:00 경과 시) 이 6칸은
  자동으로 비워집니다. 이전 당직자가 다음 당직에 따라오지 않으며, 이름은 히스토리에
  보존됩니다. 고정 라인(ER·ANE·OR·EICU·GW1·GW2)은 유지됩니다.
- 권장: 번호 칸을 길게 눌러 → 자동 완성 → 연락처 → 주소록에서 선택. 탭 시 바로 발신.
- 번호가 없는 칸은 탭하면 설정 시트가 열립니다 (v3.5.0부터 단축어 연동 없음).

## 5. 업데이트

파일 수정 후 재업로드 시 버전은 **두 곳만** 올리면 됩니다:
1. `app.js` 맨 위 `NW_VERSION` (화면 하단 표기는 여기서 자동 반영)
2. `sw.js` 맨 위 `VERSION` (캐시 갱신 트리거)

## 구조

```
index.html  화면 + 스타일 (영문 UI, 베이지, 한 화면)
app.js      당직 계산·콜 기록·CALL LINE·날씨·자동 압축
sw.js       오프라인 캐시
manifest.webmanifest / fonts/ / icons/ / test.js
```
