# Night Watch (v2.1)

밤당직(19:00 → 익일 07:00) 전용 오프라인 아이폰 웹앱. UI는 전면 영문(Lato),
베이지(사막의 낮) 테마, 스크롤 없는 한 화면, 1.5배 세로의 가로 배터리 게이지.

## 1. GitHub Pages 배포

1. github.com → New repository (예: `nightwatch`) → Create
2. Add file → Upload files → 이 폴더 안의 모든 파일·폴더 업로드
3. Settings → Pages → Branch `main` / `(root)` → Save
4. `https://<아이디>.github.io/nightwatch/` 확인

이미 배포했다면 덮어쓰기만 하면 됩니다. sw.js VERSION이 v2.1.0이라 자동 갱신됩니다.

> 참고: app.js 기본값에 들어 있습니다.


## 2. 아이폰 설치

Safari 접속(첫 1회 온라인) → 공유(⬆︎) → **홈 화면에 추가**. 이후 오프라인 완전 동작,
날씨만 온라인 시 갱신.

## 3. 화면 (한 화면, 스크롤 없음)

- 가로 배터리: 왼쪽 끝 NOW(병원 현재 날씨) → 오른쪽 끝 END(종료 시각 날씨),
  중앙에 소화율 %. 아래 ELAPSED / REMAINING.
- CALL +1: 배터리 바로 아래. ⟲ 실수 취소, ≡ 콜 시각 로그·조기 종료.
- 문구 카드: 시간대별 응원 문구(듄 + 병원 유머, 영문).
- CALL LINE: GW1 · GW2 · ICU · PA · ANE · ER · OR. 탭 = 즉시 발신.
  **전화번호는 화면에 표시되지 않습니다** (이름/라벨만).
- 작은 기종은 자동 압축(문구는 최후 단계까지 유지)으로 항상 한 화면.

## 4. CALL LINE 설정 (EDIT)

고정 번호(기본 등록): ANE 010-3079-8352 · ER 031-787-3001 · OR 031-787-3355

GW1 · GW2 · ICU · PA (당직마다 교체):
- 권장: 번호 칸을 길게 눌러 → 자동 완성 → 연락처 → 주소록에서 선택. 탭 시 바로 발신.
- 이름만 입력 시: 아래 단축어가 주소록에서 이름을 찾아 발신.

### 단축어 (선택, 1분)
단축어 앱 → ＋ → ① 연락처 검색(이름이 '단축어 입력' 포함, 1개 제한)
→ ② 전화 걸기(검색 결과) → 이름 **DutyCall** 로 저장.

## 5. 업데이트

파일 수정 후 재업로드 시 sw.js의 VERSION 숫자를 올릴 것 (예: v2.1.1).

## 구조

```

sw.js       오프라인 캐시
manifest.webmanifest / fonts/ / icons/ / test.js
```
